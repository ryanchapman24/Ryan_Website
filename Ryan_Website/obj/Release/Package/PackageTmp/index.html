Bug Tracker

This project was our largest and most ambitious project in the class in terms of scope and scale. We used DataTables for support tickets and a fully functioning notification system. The Bug Tracker is a web application that can be used for almost any company in today's web-connected business environment.

This was my first chance to break down site access in terms of what you can do as an individual based on your allocated role. The Bug Tracker utilizes the hierarchical structure of an Administrator, Project Manager, Developer, and Submitter. Administrators assign roles to users and have no restrictions in terms of site functions that are available within the application. Project Managers create and manage projects, and assign Developers to these projects. Developers are the primary worker bees for the application. The application is designed mainly for the Developers, for this reason. Developers receive email notifications and site notifications for each and every change made to one of their tickets. All new registered users are given a Submitter role upon registration. Submitters are the ones finding "bugs" within the software, then going to the site and submitting tickets for these bugs. Developers then are assigned to these tickets and they manage them from inception to completion.

Come check out the web-app! You can register, or demo the site for each role via the demo buttons on the login page!

Budgeting System

This is the web application that I'm most proud of in terms of the front-end styling. As a web developer, you have to demonstrate your ability to make the user experience as seamless and as enjoyable as possible. Budgeting can be a stressful task, so my goal was to create a fun environment for the user. This was also my first time using charts and graphs to visually represent constantly changing data.

Users create Households and then subsequently can invite other users to their Households via email invitations. Once you are part of a Household, you add your bank accounts that you will be using to manage your budgets that you create. From anywhere on the site, users can add a new transaction for a specific account, which dynamically updates the account balances, budget allocation, and household performance. Users can view a list of all transactions, or they can view a list of account specific transactions. From there, they are able to reconcile their accounts to make sure they match up with what reads on their bank statements.

Visit the site to create your very own Household! Have some fun with it. Invite friends and family and see what possibilities ensue!

                                    $(document).ready(function () {
                                        $(".input-f").focus(function () {
                                            $(this).css("outline-color", "#388E8E");
                                        });

                                        $("#submit-f").click(function () {
                                            var one = parseInt($("input[name=1st-f]").val());
                                            var oneF = parseFloat($("input[name=1st-f]").val());
                                            var factorial = 1;
                                            if ($.trim($("#numbers-f").text()) != "") {
                                                $("#numbers-f").empty();
                                            }

                                            if (isNaN(one)) {
                                                $("#error-f").empty();
                                                $("#error-f").append("
You must submit a number!
");
                                            }

                                            else if (one < 0) {
                                                $("#error-f").empty();
                                                $("#error-f").append("
Your number must be positive!
");
                                            }

                                            else if ((oneF % 1) != 0) {
                                                $("#error-f").empty();
                                                $("#error-f").append("
You must enter a whole number!
");
                                            }

                                            else if (one === 1) {
                                                $("#error-f").empty();
                                                $("#numbers-f").append("
1! = 1.
");
                                            }

                                            else if (one === 0) {
                                                $("#error-f").empty();
                                                $("#numbers-f").append("
0! = 1..... for some odd reason...
");
                                            }

                                            else {
                                                if (one >= 0) {
                                                    $("#error-f").empty();
                                                    for (i = one; i > 1; i--) {
                                                        factorial *= i;
                                                    }
                                                    $("#numbers-f").prepend("
" + one + "! = " + factorial + ".
");
                                                }
                                            }
                                        });

                                        $("#again-f").click(function () {
                                            $("#numbers-f").empty();
                                            $("#error-f").empty();
                                            $("#form-f")[0].reset();
                                        });
                                    });
